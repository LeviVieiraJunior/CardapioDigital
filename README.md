# Projeto de Pedidos Online

Este é um sistema de pedidos online completo, com frontend em React e backend em Node.js.

## Estrutura do Projeto

- `frontend/`: Aplicação web para usuários e painel administrativo.
- `backend/`: API e lógica de negócios.
- `DOCUMENTACAO_MANUTENCAO.txt`: Guia detalhado de manutenção do projeto.

## Como Iniciar

1. Clone o repositório.
2. Siga as instruções de instalação no `DOCUMENTACAO_MANUTENCAO.txt`.
